# CSV Analysis Tool

https://csv-analysis-app.netlify.app/<br>
A polyglot SaaS website that allows users and businesses to get professional-grade CSV analysis at the click of a button.

## Current Version Tech Stack

Frontend: React, TypeScript, Bootstrap, Vite<br>
Backend: Node.js, Python, Express (With Multer)<br>
Auth: Clerk<br>
AI: Google Gemini API<br>

## Challenges Faced

In this project, I faced many challenges, which included:<br>

- Minimising AI Token usage<br>
- Securing backend endpoints with Clerk middleware to prevent token abuse<br>
- Managing a large project on my own<br>
- Strictness of TypeScript<br>
- Ensuring no user data is ever stored server-side for longer than necessary<br>
